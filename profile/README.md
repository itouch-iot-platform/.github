## 📦 Organization Content

This organization contains the complete **source code ecosystem** of the ITouch platform, split into multiple independent services.

Each repository represents a microservice or application component of the overall system.

---

## 🗂️ Repository Structure

| Repository | Description |
|------------|-------------|
| `ssio-project-full-backend` | Main backend API handling business logic and data management |
| `ssio-project-gateway` | API Gateway acting as the entry point to the system |
| `SSIO-Project-FrontEnd` | Web dashboard for visualization and user interaction |
| `SSIO-Project-Mobile` | Mobile application for accessing the IoT platform |
| `IDS-Server` | Intrusion Detection System for monitoring and security analysis |
| `Flow_Generator_Server` | Service responsible for generating simulated IoT data streams |

---

## ⚙️ Architecture Overview

The ITouch platform follows a **distributed microservices architecture**:

- 🧠 Backend services manage core logic and data processing  
- 🌐 Gateway provides a unified entry point and request routing  
- 📊 Frontend offers real-time visualization and control  
- 📱 Mobile application enables remote access to the platform  
- 🛡️ IDS monitors system traffic and detects anomalies  
- 🔄 Flow Generator simulates IoT data streams for testing and evaluation

## 🚀 Project Launch

To run the entire project:

1. Extract the archives from `8- Code Source/`.
2. Go to the full backend repository (archive `ssio-project-full-backend-main.zip` once extracted).
3. Start the project using Docker.

The ITouch platform aims to provide a **scalable and modular IoT ecosystem**, combining:

- Real-time data processing
